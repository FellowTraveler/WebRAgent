About WebRAgent

What is WebRAgent?

WebRAgent is a powerful Retrieval-Augmented Generation (RAG) web application built by Dennis Kruyt. This advanced system combines the capabilities of Large Language Models (LLMs) with a vector database (Qdrant) to provide context-enhanced responses to your queries.

Unlike traditional search engines or chatbots, WebRAgent can search and analyze specific document collections or the web, using sophisticated AI techniques to provide comprehensive, relevant, and accurate answers.

Key Features

  • Collection Search: Query your document collections
  • Web Search: Find information across the internet
  • Agent Search: Break down complex questions
  • Mind Maps: Visualize concepts and relationships
  • Deep Web Search: Analyze full web page content
  • Multiple LLM Support: OpenAI, Claude, Ollama
  • Document Processing: Automatic chunking & embedding
  • Admin Interface: Manage collections and documents

Technology Stack

WebRAgent is built using modern technologies and open-source frameworks:

  • Python & Flask: Web application framework
  • Qdrant: Vector database for embeddings
  • SearXNG: Privacy-respecting metasearch engine
  • Large Language Models: OpenAI, Claude, Ollama
  • SentenceTransformers: For embeddings generation
  • Bootstrap: For responsive UI

About the Creator

WebRAgent was created by Dennis Kruyt, a software engineer and AI enthusiast passionate about developing practical applications that leverage the power of modern AI technologies.

This project represents a comprehensive implementation of Retrieval-Augmented Generation technology in a user-friendly web interface, designed to make advanced AI capabilities accessible and useful.
